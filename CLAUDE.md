# 薬剤師たぬぽん ポートフォリオサイト — 編集方針

## サイト概要

現役薬剤師「たぬぽん」のポートフォリオ。ターゲットはコンテンツ制作を依頼したい医療メディア・クリニック・企業。
ココナラで薬機法スクリーニング・医療記事執筆を受注するための実績・信頼提示が主目的。

## コンテンツ編集の基本ルール

- **コンテンツは `src/config.ts` の `siteConfig` を編集する**。CMS・Firestore はない。
- 編集後は必ず `git add → commit → push` まで実行する（pushしないとNetlifyに反映されない）。

## トーン・ポジショニング方針

### 薬剤師専門性を主役に、AIは後景へ

医療業界のクライアントはAIへの警戒感が強い。以下を守ること：

- **NG**: 「AI × 薬剤師」「AIで爆速」「AIスペシャリスト」など、AIを前面に押し出す表現
- **OK**: 「薬剤師8年の専門知識」「薬機法準拠」「一次情報確認」「責任を持って監修」
- AIに触れる場合は「補助ツールとして活用、判断は薬剤師が行う」という形にとどめる

### 守るべき既存コピー

About セクションの以下の文章は差別化として機能しているので変更しない：
> 「Claude・NotebookLMなどのAIは情報収集・構成補助に活用しますが、医学的判断・薬機法確認・最終監修は薬剤師として責任を持って行います。」

### 文体

- 本文：丁寧で信頼感のある硬派な文体
- CTA・リード：親しみやすさを出してよい（キャラクター「たぬぽん」感）
